# Tlanner 

## O que é o Tlanner?

Tlanner é um task manager/planner simples e prático, onde o usuário pode registrar tarefas em um kanban board para ter controle da suas tarefas e rotinas.

http://localhost:3000/api-docs/

## Ferramentas Utilizdas

Mysql, Express, Cors, Docker, Swagger
